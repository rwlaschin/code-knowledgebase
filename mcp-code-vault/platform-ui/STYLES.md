# UI styles

**Tailwind CSS** is used for all styling. Use these patterns so the app matches the intended look.

- **Page:** Dark background, light text — `min-h-screen bg-gray-900 text-gray-100 p-8`
- **Page title:** `text-3xl font-bold mb-6`
- **Nav:** `flex gap-4 mb-8`; links `text-purple-400 hover:underline`
- **Cards/sections:** `bg-gray-800 rounded-lg p-6`
- **Section heading:** `text-xl font-semibold mb-2`
- **Muted text:** `text-gray-400`
- **Scrollable/pre:** `text-sm overflow-auto max-h-40`

Result: dark theme (gray-900/gray-800), purple accent links, rounded cards. No white background or default blue links.

# Platform UI

Nuxt 3 app for stats, config, and docs. Uses Tailwind CSS (via `@nuxtjs/tailwindcss`).

## Run the dev server

**From this directory** (required so Nuxt can load `@nuxtjs/tailwindcss` from `node_modules`):

```bash
cd mcp-code-vault/platform-ui
npm install   # if you haven't already
npm run dev
```

Or from the repo root:

```bash
npm run dev:ui
```

If you see "Could not load @nuxtjs/tailwindcss", you're not running from `platform-ui`; use one of the options above.
